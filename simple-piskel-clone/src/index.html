Piskel Clone

Piskel Clone is an online editor for animated sprites.

Application functionality

  • Tools: pen, eraser, color select (primary/ secondary), color picker, paint bucket (closed area and all pixels of the same color).
  • Ð¡anvas scaling: 32x32 / 64x64 / 128x128 units
  • Frames: ability to add, delete, duplicate.
  • Preview: ability to animate the created frames, full screen mode.
  • Keyboard shortcuts
  • Log in: via google account
  • Export: as animated gif
